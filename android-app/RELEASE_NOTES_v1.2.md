# Quest Home Switcher 1.2

This release combines a direct rooted environment picker with the existing rootless Shizuku workflow.

## What's new

- Automatic Magisk/`su` detection
- Installed environment discovery on rooted Quest devices
- Direct `oculuspreferences` switching
- Automatic VR Shell reload; normally no reboot in root mode
- Rootless fallback through Shizuku
- No required `Quest Homes` folder: APKs in `Download`, `QuestHomes`, `Homes`, or subfolders are found automatically

Home APKs are not bundled. This is an unofficial community project and is not affiliated with Meta, Shizuku, Quest Home Porter, or xAstroBoy.
