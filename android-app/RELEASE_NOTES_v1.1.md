# Quest Home Switcher 1.1

Version 1.1 is a stability update that returns to the proven behavior of version 1.0.

## Changes

- Restores rootless Home switching without a required Quest restart.
- Restores normal minimize and background behavior.
- Detects and marks the currently installed Home.
- Adds a lightweight layered-glass depth effect to the Home list.
- Removes the experimental transition and background-card changes.
- Uses the stable build toolchain from version 1.0.

## Requirements

- Meta Quest 2, Quest 3, or Quest Pro
- Developer Mode
- Shizuku 13.x started through Wireless ADB

This is an unofficial community project and is not affiliated with Meta, Shizuku, or Quest Home Porter. Home APKs are not included. Use at your own risk.
